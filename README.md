 Alinma Insights

 Overview
Alinma Insights is an AI-powered financial intelligence center integrated into the Alinma Bank mobile application. It transforms raw banking transactions into personalized financial insights using Generative AI.

Features

- AI Financial Dashboard
- Smart Transaction Categorization
- AI Financial Assistant
- Spending Analytics
- Subscription Center
- Linked Merchants
- Monthly Wrapped
- Personalized AI Insights

 Tech Stack

- Lovable
- React
- TypeScript
- Tailwind CSS
- Recharts
- JSON Dataset
- Generative AI
- Open Banking (Future Integration)

Dataset

Synthetic banking dataset representing six months of realistic Saudi banking activity.

Future Work

- Open Banking integration
- Live banking APIs
- Real AI model integration
- Push notifications
